# Helo Gourmet Delícias

Este projeto é uma aplicação web para a Helo Gourmet, que oferece uma variedade de sacolés gourmet. A aplicação é responsiva e otimizada para diferentes tamanhos de tela, proporcionando uma experiência de usuário agradável em dispositivos móveis e desktops.

## Estrutura do Projeto

O projeto possui os seguintes arquivos:

- **index.html**: Este arquivo é a estrutura principal da aplicação. Ele contém a marcação HTML, incluindo cabeçalho, menu, seções de produtos, pedidos e redes sociais. O arquivo também inclui links para folhas de estilo e scripts.

- **css/style.css**: Este arquivo contém os estilos CSS para a aplicação. Ele define variáveis de cores, estilos gerais, estilos para a barra superior, menu, botões, slideshows e responsividade para diferentes tamanhos de tela.

- **js/script.js**: Este arquivo é destinado a scripts JavaScript que podem ser usados para adicionar interatividade à aplicação, como a navegação entre slides e a manipulação do menu.

- **img/icon/icons8-facebook-48.png**: Este arquivo é um ícone do Facebook utilizado na seção de redes sociais.

- **img/icon/icons8-ifood-96.png**: Este arquivo é um ícone do iFood utilizado nos botões de pedido.

- **img/icon/icons8-instagram-96.png**: Este arquivo é um ícone do Instagram utilizado na seção de redes sociais.

## Como Configurar e Executar a Aplicação

1. **Clone o repositório**:
   ```
   git clone <URL do repositório>
   ```

2. **Navegue até o diretório do projeto**:
   ```
   cd helogormetnew
   ```

3. **Abra o arquivo `index.html` em um navegador**:
   Você pode abrir o arquivo diretamente no navegador ou usar um servidor local para visualizar a aplicação.
